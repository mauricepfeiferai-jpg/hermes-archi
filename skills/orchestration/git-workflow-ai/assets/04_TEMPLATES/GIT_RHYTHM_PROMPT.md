# Git Rhythm Prompt for Claude Code

```text
I want to update my [SKILL/CLAUDE.md/eval criteria].

1. Pull the latest changes from main.
2. Create a new branch called [improve/name-vN].
3. Apply this change: [describe change].
4. Commit with the message: [specific change description].
5. Push my changes.
6. Open a pull request.
```

Solo workspace? Skip branch and PR, commit directly on main.

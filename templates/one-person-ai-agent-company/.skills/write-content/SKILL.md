# Write Content

## Inputs

- `researcher/outputs/daily_brief_YYYY-MM-DD.md`
- `ceo/outputs/daily_goals_YYYY-MM-DD.md`
- `youtube_operator/outputs/content_calendar_*.md` (optional)

## Output

- `writer/outputs/daily_content_YYYY-MM-DD.md`

## Definition of Done

- [ ] One concrete content piece written
- [ ] Uses numbers, not adjectives
- [ ] One clear CTA
- [ ] Tone matches brand voice guide
- [ ] Human approval required before posting

# Job Description: YouTube Operator Agent

## Purpose
Run a YouTube channel as a system, not a creative guessing game.

## Responsibilities
1. Find and validate niches using TAM + passion framework
2. Maintain a brand voice guide for the channel
3. Generate a monthly content calendar from proven outliers
4. Write titles, thumbnails concepts, script outlines, and hooks
5. Track analytics and recommend improvements
6. Draft sponsorship outreach and monetization offers

## Inputs
- `researcher/outputs/daily_brief_*.md` (trends)
- Brand voice memo / transcript
- YouTube analytics screenshots
- Previous video performance data

## Outputs
- `youtube_operator/outputs/niche_validation_*.md`
- `youtube_operator/outputs/brand_voice_guide.md`
- `youtube_operator/outputs/content_calendar_*.md`
- `youtube_operator/outputs/video_briefs_*.md`
- `youtube_operator/outputs/analytics_review_*.md`
- `youtube_operator/outputs/sponsor_pitch_*.md`

## Key Prompts
1. Niche finder: "i'm thinking of starting a youtube channel about [X]. what are five problems i can solve and how big is the audience"
2. Brand voice: "write a brand voice guide for my channel about [niche]. tone, phrases to use/avoid, setting, colors, wardrobe, core philosophies"
3. Titles: "write 10 title variations, score each 1-10, recommend top three"
4. Thumbnails: "suggest three thumbnail concepts, visual composition, three-word text max, color palette"
5. Script outline: "write a bullet-pointed script outline, hook 30s max"
6. Hooks: "write three hook variations, bold claim / relatable pain / surprising fact"
7. Pinned comment: "write a pinned comment that asks a question to increase engagement"
8. Analytics review: "analyze these YouTube analytics screenshots and recommend next steps"
9. Monetization: "how do I monetize this channel, what to charge, draft launch video + landing page"

## Schedule
- Weekly: content calendar + outlier validation
- Per video: titles, thumbnails, script outline, hooks
- Monthly: analytics review + monetization update

## Quality Gates
- [ ] Every video idea has 4+ backing outliers in last 6 months
- [ ] Brand voice guide exists and is used
- [ ] Script outlines are bullet points, not full sentences
- [ ] Every video has 3 title + 3 thumbnail + 3 hook options

## Tools
- Claude project / Claude Code
- 1of10 Chrome extension
- YouTube burner account strategy
- Canva / Photopea / Photoshop for thumbnails
- `post-x` skill for final posts

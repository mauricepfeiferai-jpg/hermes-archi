# YouTube Operator Agent

A complete YouTube channel operating system using Claude as strategist.

## The 5 Layers

1. Niche — passion + TAM
2. Brand — voice, visuals, wardrobe, philosophy
3. Ideas — burner account + outlier validation
4. Packaging — titles, thumbnails, scripts, hooks
5. Scale — post checklist, analytics, monetization

## How to Use

1. Fill in your niche idea in `youtube_operator/outputs/niche_validation_*.md`
2. Record a 5-minute voice memo about your unique beliefs and feed it to the brand voice prompt
3. Set up a burner YouTube account for idea generation
4. Run the content calendar prompt weekly
5. Use the per-video prompts for every upload
6. Monthly analytics review with Claude

## Monetization

- Ad revenue (1k subs + 4k watch hours)
- Sponsorships
- Own product/service

# Product Factory Agent: Web Builder

## Role

Generate a single-file HTML landing page from product copy.

## Design System

- Background: #0a0a0a
- Text: #ffffff
- Accent: #00d4ff
- Max-width: 680px for product pages, 900px for index
- Font: system sans-serif
- One CTA button per page

## Output

- `website/products/{slug}.html`
- Must include: headline, subheadline, problem, solution, proof, pricing, CTA, footer.
